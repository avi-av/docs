Update¶

class pyrogram.Update(message=None, edited_message=None, deleted_messages=None, channel_post=None, edited_channel_post=None, deleted_channel_posts=None, inline_query=None, chosen_inline_result=None, callback_query=None, shipping_query=None, pre_checkout_query=None)¶

This object represents an incoming update. At most one of the optional parameters can be present in any given update.

Parameters:
  • message (Message, optional) – New incoming message of any kind — text, photo, sticker, etc.
  • edited_message (Message, optional) – New version of a message that is known to the bot and was edited.
  • deleted_messages (Messages, optional) – Deleted messages.
  • channel_post (Message, optional) – New incoming channel post of any kind — text, photo, sticker, etc.
  • edited_channel_post (Message, optional) – New version of a channel post that is known to the bot and was edited.
  • deleted_channel_posts (Messages, optional) – Deleted channel posts.
  • inline_query (InlineQuery, optional) – New incoming inline query.
  • chosen_inline_result (ChosenInlineResult, optional) – The result of an inline query that was chosen by a user and sent to their chat partner. Please see our documentation on the feedback collecting for details on how to enable these updates for your bot.
  • callback_query (CallbackQuery, optional) – New incoming callback query.
  • shipping_query (ShippingQuery, optional) – New incoming shipping query. Only for invoices with flexible price.
  • pre_checkout_query (PreCheckoutQuery, optional) – New incoming pre-checkout query. Contains full information about checkout.
